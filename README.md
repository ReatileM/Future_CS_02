Phishing Detection & Awareness Report

Overview
This project demonstrates real-world phishing email analysis using practical examples.

Objectives
- Identify phishing indicators
- Analyze suspicious emails
- Classify risk levels
- Educate users on phishing prevention

Case Studies
- Recruitment Scam Email
- iCloud Payment Phishing Email

Tools Used
- Google Admin Toolbox (Header Analyzer)
- MXToolbox
- Manual email inspection

Methodology
1. Collected phishing samples
2. Analyzed sender and content
3. Identified red flags
4. Classified risk
5. Documented findings

Outcome
This project helps improve cybersecurity awareness and reduce phishing risks.
